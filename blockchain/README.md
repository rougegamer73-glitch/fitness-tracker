# FitChain Blockchain

Hardhat workspace for the FitChain smart contracts.

## Setup

```shell
npm install
```

## Run Local Hardhat Node

```shell
npm run node
```

The local Hardhat network runs at `http://127.0.0.1:8545` with chain id `31337`.

## Useful Commands

```shell
npm test
npm run deploy
npx hardhat compile
```

Compiled contract artifacts are written to `../frontend/src/artifacts` for the frontend.
